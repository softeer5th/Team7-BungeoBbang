## ON:U Frontend

### Directory Structure

```
frontend/
├── public/
├── src/
│ ├── assets/
│ │ ├── icons/
│ │ └── imgs/
│ ├── components/ # 공통 UI 컴포넌트
│ ├── config/ # 환경 설정 (API URL, 서브 도메인 매핑)
│ ├── domains/ # 학생 / 학생회 도메인 구현 폴더
│ ├── hooks/ # 전역 커스텀 훅
│ ├── routes/ # 라우팅 관련
│ ├── store/ # 상태 관리
│ ├── styles/ # 전역 스타일 관련
│ ├── utils/ # 유틸리티 함수
│ ├── App.css
│ ├── App.tsx
│ ├── index.css
│ ├── main.tsx

```
